## 项目描述

一个面向高校学生的二手交易平台，支持商品发布、即时聊天、校园认证等功能。

## 技术栈

- Vue.js 3 + TypeScript
- Node.js + Express
- MongoDB 数据库
- WebSocket 即时通讯

## 主要成果

- 日活用户1000+
- 支持实时聊天功能
- 实现校园邮箱认证
