## 项目描述

支持多人实时协作的在线白板工具，集成绘图、批注、PPT演示功能。

## 技术栈

- React + TypeScript
- Socket.io 实时同步
- HTML5 Canvas
- Redis 缓存

## 主要成果

- 支持50人同时在线协作
- 实时同步延迟<100ms
- 集成Microsoft Office在线编辑
