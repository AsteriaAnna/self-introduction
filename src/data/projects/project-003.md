## 项目描述

基于AI的日程管理助手，支持智能提醒、日程冲突检测、自然语言输入。

## 技术栈

- React 18 + TypeScript
- Python FastAPI
- SQLite 数据库
- OpenAI API

## 主要成果

- 日程冲突检测准确率95%
- 自然语言解析支持10+种表达
- 日均提醒次数5000+
