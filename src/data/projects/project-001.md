## 项目描述

这是一个现代化的个人展示网页平台，支持项目展示、经历展示、JD智能匹配和知识图谱可视化。

## 技术栈

- React 18 + TypeScript
- Vite 构建工具
- Tailwind CSS 样式
- D3.js 知识图谱
- React Router 路由

## 主要成果

- 实现响应式设计
- 支持明暗主题切换
- 实现JD关键词匹配功能
